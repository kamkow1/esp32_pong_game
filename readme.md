# esp32 pong game


## how to build

### dependencies
in order to build this project, you'll have to have ESP32 IDF installed.

[official docs](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/index.html)

- building the server:
```
cd game_server
idf.py build
```
